Ejercicio 2

Usando la librería Javascript de tu preferencia, realiza una petición Ajax a la siguiente URL https://swapi.co/api/people/, los datos que devuelve dicha API son los nombres de los personajes de Star Wars.

Con los datos que obtengas, deberás realizar lo siguiente:

  • Avatar o icono del personaje (usar http://starwarsglyphicons.com/#getstarted)
  • Nombre
  • Altura
  • Color de Cabello
  • Cumpleaños
  • Nombre de su planeta de origen (se deberá hacer una petición Ajax adicional).

La maquetación se deberá hacer sin uso de frameworks, usar etiquetas semánticas donde consideres conveniente, debe soportar resoluciones desde 320px en adelante y verse bien en Google Chrome, Firefox y Microsoft Edge.
